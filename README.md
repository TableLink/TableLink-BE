## 프로젝트 소개

### 프로젝트 기간 및 인원

- 기간: 2024.10.06 - 2024.11.
- 인원: 1명(개인 프로젝트)

### 프로젝트 개요

- **프로젝트 이름**: TableLink
- **프로젝트 초기 목표**: JWT 방식의 인증 인가 로직을 구축하고 음식점을 예약할 수 있는 서비스를 개발 및 음식점에 대한 리뷰를 남기고 자유롭게 대화할 수 있는 커뮤니티
  플렛폼 개발
- **축소된 프로젝트 목표**: JWT 방식의 인증 인가 로직을 구축하고 Kakao Map API를 활용하여 음식점에 대한 리뷰를 남기고 자유롭게 대화를 나누며 나만의 맛집
  지도를 공유할 수 있는 커뮤니티 사이트 개발
